🎯 Čo je metóda bisekcie?

Metóda bisekcie je numerická metóda na hľadanie koreňa (nulového bodu) spojitej funkcie. Je založená na vete o strednej hodnote.

📐 Princíp fungovania

Ak máme interval [a, b], kde f(a) a f(b) majú rôzne znamienka, potom v tomto intervale musí existovať koreň.

💡 Kľúčová myšlienka: Ak funkcia mení znamienko, musí prejsť cez nulu!

🔄 Algoritmus

  1. Vypočítaj stred: c = (a + b) / 2
  2. Vyhodnoť: f(c)
  3. Kontrola presnosti: Ak |b - a| ≤ 2·tolerancia → KONIEC ✅
  4. Rozhodnutie:
    • Ak f(a) · f(c) < 0 → koreň v [a, c]
    • Inak → koreň v [c, b]
  5. Opakuj
⚠️ Dôležité: Metóda bisekcie kontroluje dĺžku intervalu |b - a|, nie hodnotu funkcie |f(c)|!
Keď |b - a| ≤ 2·tolerancia, koreň je určený s presnosťou ±tolerancia.

📊 Vzorce

c = (a + b) / 2
Zastaviť ak: |b - a| ≤ 2·tolerancia
f(a) · f(c) < 0 → nový interval [a, c]
f(a) · f(c) > 0 → nový interval [c, b]

📏 Odhad chyby

Po n iteráciách je chyba aproximácie koreňa ohraničená vzorcom:

|cₙ − α| < (b₀ − a₀) / 2ⁿ⁺¹

kde α je skutočný koreň, b₀ − a₀ je dĺžka počiatočného intervalu.

⚡ Rýchlosť konvergencie

Metóda bisekcie má lineárnu konvergenciu - každý krok zmenší interval presne na polovicu:

|bₙ − aₙ| = (b₀ − a₀) / 2ⁿ
