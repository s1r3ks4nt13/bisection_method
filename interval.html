🎯 Podmienka: f(a) · f(b) < 0

Funkcia musí meniť znamienko na intervale [a, b]!

To znamená, že v intervale musí byť aspoň jeden bod, kde funkcia má nulovú hodnotu (pretína os X). Ak f(a) a f(b) majú opačné znamienka, funkcia určite prechádza cez nulu v tomto intervale.

f(a) · f(b) < 0
💡 Bolzanova veta: Ak je funkcia spojitá na [a, b] a f(a) · f(b) < 0, potom existuje aspoň jeden koreň c ∈ (a, b) taký, že f(c) = 0.

🔍 Ako nájsť správny interval?

Hľadaj dve hodnoty a a b také, že funkcia zmení znamienko:

f(a) < 0 a f(b) > 0
alebo: f(a) > 0 a f(b) < 0
⚠️ Pozor: Ak f(a) · f(b) > 0 (rovnaké znamienka), interval je neplatný - metóda bisekcie sa nedá použiť! Buď interval neobsahuje koreň, alebo obsahuje párny počet koreňov.

🎓 Systém podpôr:

  • Pokus 1: Skús sám - zadaj hodnoty a a b bez pomoci
  • Pokus 2: Graf dvoch funkcií - graf rozdelený na dve časti ukáže, kde sa funkcie pretínajú, čo zodpovedá koreňu rovnice
  • Pokus 3: Kompletný graf funkcie - vidíš presné miesta, kde funkcia pretína os X
